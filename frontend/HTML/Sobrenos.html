GreenSea - Sistema de Controle de Pragas

Sobre o Sistema

O sistema GreenSea foi desenvolvido para auxiliar empresas especializadas em controle de pragas no gerenciamento de clientes e serviços prestados.

Com ele, é possível cadastrar clientes, registrar usuários da equipe, organizar os atendimentos realizados e acompanhar os dados de forma clara e centralizada. O sistema também permite a visualização de informações de pagamento e o acesso a painéis administrativos para melhor controle das atividades.

Acesso ao Sistema

Login padrão para primeiro acesso:

  • Usuário: Admin
  • Senha: 1234

Após o login, o usuário administrador poderá acessar a seção de configurações e cadastrar outros usuários, conforme necessário.
